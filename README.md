# CardioScope — Heart Disease Prediction System

## Overview

CardioScope is an interactive machine learning web application built with Streamlit for heart disease prediction. The application uses a Random Forest Classifier trained on clinical patient data to predict the likelihood of heart disease.

## Features

* Heart disease risk prediction
* Interactive dashboard
* Model performance visualization
* Feature importance analysis
* Dataset insights and statistical analysis
* Real-time prediction interface

## Technology Stack

* Python
* Streamlit
* Scikit-learn
* Pandas
* NumPy
* Plotly

## Dataset

The project uses the Heart Disease dataset containing clinical and demographic patient information.

## Model

The prediction model is based on a Random Forest Classifier trained using an 80/20 train-test split.
